# Products App

A simple static product browser that fetches data from the Fake Store API and stores it in local storage.

## GitHub Pages deployment

1. Create a GitHub repository and push this project.
2. In repository settings, enable GitHub Pages.
3. Set the source to **GitHub Actions**.
4. The workflow in [.github/workflows/deploy.yml](.github/workflows/deploy.yml) will publish the site automatically on every push to `main`.
